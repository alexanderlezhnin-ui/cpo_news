🔴 Угрозы
Угроза #1
Bnovo: двухсторонний API + программа лояльности
Bnovo запустил API «Профессионал» с полным CRUD (GET+PUT+POST+вебхуки) и встроенную программу лояльности с автопромокодами по номеру телефона. Почему это угроза: API — ключевой инструмент для интеграторов и крупных сетей. Если Bnovo закроет этот сегмент раньше, TL потеряет потенциальных enterprise-клиентов. Программа лояльности — retention-инструмент, которого у TL нет из коробки.
@bnovonews/3682 · лояльность
Угроза #2
Контур.Отель в «белом списке» Минцифры
Контур.Отель включён в реестр ПО, работающего при ограничениях связи. Почему это угроза: в регионах с нестабильным интернетом (курорты, удалённые локации) это реальное конкурентное преимущество. Отельеры обсуждают надёжность системы в чатах. Если TL не в списке — это аргумент против при выборе.
@wrkhotel/5152 · @otelkontur

Ценовой барьер при переходе с Bnovo
«Считаю удорожание расходов при переходе с бново на ТЛ» — отельер Anna F в Беседке. Почему это угроза: клиенты Bnovo — естественный источник роста для TL, но если переход воспринимается как дорогой, конверсия страдает. Нужна чёткая коммуникация ROI и программа миграции.
